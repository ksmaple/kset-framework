# Engineering 域规范

G001: 先建模被控对象、控制量、被控量、扰动量、参考输入
G002: 策略实施前须至少 7 天基线 metrics
G003: 稳定性等于熔断、降级、探针与服务分级 P0-P3
G004: 性能调优优先反馈控制，非固定配置
G005: HPA/VPA 识别瓶颈，二者不同时启用
G006: CI/CD 作质量反馈回路，门禁自动化
G007: 成本优化不得牺牲 P0 SLA
G008: 控制参数集中注册、可配置、可观测
G009: 回路失效回退保守默认值
G010: 禁止正反馈；单次调整不超过 20%
G011: CI 门禁：编译、单测覆盖率大于 80%、Sonar 阻断、集成、性能回归
G012: 灰度 5%→10%→50%→100%，各阶段观察指标与时长
G013: 不产出 design/control 文档；落地看配置与监控代码
